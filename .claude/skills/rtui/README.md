# rtui - 交互式终端界面

## 概述

`rtui` 是一个基于 ratatui 的交互式终端界面工具，提供可视化的服务和环境变量管理。

## 构建

```bash
cargo build --features tui --bin rtui
cargo run --features tui --bin rtui
```

## 功能

### 服务管理标签页

- 列出所有已安装的服务
- 使用方向键或 j/k 导航
- 按 r 刷新服务列表
- 按 / 搜索过滤服务
- 按 e 编辑选中的服务配置

### 环境变量标签页

- 列出所有用户级别环境变量
- 彩色显示键值对
- 使用方向键或 j/k 导航
- 按 r 刷新环境变量列表
- 按 / 搜索过滤环境变量
- 按 e 编辑选中的环境变量值

## 快捷键

| 按键 | 功能 |
|------|------|
| Tab | 切换标签页 |
| ↑/k | 向上移动 |
| ↓/j | 向下移动 |
| r | 刷新当前列表 |
| / | 进入搜索模式 |
| e | 编辑选中项 |
| q | 退出程序 |

### 搜索模式

- 输入字符进行实时过滤
- Backspace 删除字符
- Enter 或 Esc 退出搜索模式

### 编辑模式

**服务编辑**：
- 调用系统编辑器（$EDITOR 或 $VISUAL）
- 编辑服务配置文件
- 保存后自动返回 TUI

**环境变量编辑**：
- 在弹出框中编辑变量值
- Enter 保存更改
- Esc 取消编辑

## 代码结构

模块化设计，便于维护：

```
src/
├── tui/
│   ├── mod.rs      # 模块导出
│   ├── app.rs      # 应用状态和业务逻辑
│   └── ui.rs       # 界面渲染
└── bin/
    └── rtui.rs     # 主程序入口
```

## 特性

- 跨平台支持（macOS、Linux、Windows）
- 实时搜索过滤
- 键盘导航
- 彩色高亮
- 状态栏提示
- 模态编辑对话框
- 外部编辑器集成

## 依赖

- ratatui: 终端 UI 框架
- crossterm: 跨平台终端操作
